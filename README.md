# IPB2 Figure Store Management System

This repository is now organized similarly to your `EmployeeLeaveManagementSystem` sample, with multiple application layers and entry points.

## Projects

- `IPB2.FigureStoreManagementSystem.DB` - EF Core database models and DbContext.
- `IPB2.FigureStoreManagementSystem.Domain` - shared domain layer.
- `IPB2.FigureStoreManagementSystem.Console` - console entry point.
- `IPB2.FigureStoreManagementSystem.MVC` - MVC web app.
- `IPB2.FigureStoreManagementSystem.MVCwithHttpClient` - MVC app prepared for API calls.
- `IPB2.FigureStoreManagementSystem.MinimalApi` - lightweight HTTP API.
- `IPB2.FigureStoreManagementSystem.WebApi` - controller-based Web API.
- `IPB2.FigureStoreManagementSystem.WinForm` - Windows Forms desktop entry point.

## Solution File

All projects are included in `IPB2.FigureStoreManagementSystem.DB.slnx`.
